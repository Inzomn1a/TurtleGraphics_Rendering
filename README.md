# Setup instructions

Clone this repo

## Setup on Centos 7

```
yum install -y python36-tkinter
pipenv --python 3.6
pipenv shell
pipenv sync
pip install colorgram.py
python main.py
```

## Setup on Mac

```
brew install python-tk
pipenv --python 3
pipenv shell
pipenv sync
pip install colorgram.py
python main.py
```
